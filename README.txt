NoteApp is a simple, full-stack note-taking application that allows users to register, log in, and manage their notes securely and efficiently.

Features:

- User registration and login using JSON Web Tokens (JWT) for authentication.
- CRUD functionality for managing notes.
- Responsive design for both desktop and mobile users.

Technologies

This project is built using:
- **Backend**: Node.js with Express
- **Database**: MongoDB
- **Frontend**: HTML, CSS, and JavaScript (with Axios for API calls)
- **Authentication**: JWT for secure access


Prerequisites

What you need to install the software:

- [Node.js](https://nodejs.org/)
- [MongoDB](https://www.mongodb.com/try/download/community)
- NPM (usually comes with Node.js)
